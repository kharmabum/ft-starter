# ft-starter

## Build Process

- Framekwork: [Ember App Kit], [eak-update](https://github.com/gevious/eak-update)
- Task Runner: [Grunt]
- Package Managers: [NPM], [Bower]
- Testing: [QUnit], [Mocha], [Karma]

## Backend

- Db: [MongoDb]
- ORM: [Mongoose]
- Server: [Express]

## Frontend

- Client: [Ember] + [Ember Data]
- Templates: [Handlebars]
- Styles: [Bootstrap]

## Build
```
  > npm install
  > bower install
  > grunt server
```
